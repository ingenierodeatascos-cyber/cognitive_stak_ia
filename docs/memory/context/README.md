# MEMORY CONTEXT

Usa esta carpeta para guardar contexto estable y reusable.

Ejemplos validos:

- reglas de dominio que se repiten
- restricciones de una integracion externa
- convenciones del proyecto que conviene recordar

No guardes aqui:

- notas temporales
- tareas puntuales
- resumenes largos de cambios concretos
- brainstorming
