# MEMORY PATTERNS

Guarda aqui patrons validats i repetibles.

## Guarda aqui

- formes repetibles de crear, editar o validar dades
- patrons de formularis
- patrons de control d'errors

## No guardis aqui

- pegats puntuals
- solucions temporals
- historic detallat d'un canvi

## Exemple bo

“Per crear una tasca: validar input, validar usuari, guardar i retornar resposta controlada.”
