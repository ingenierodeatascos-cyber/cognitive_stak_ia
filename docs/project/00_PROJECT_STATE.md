# PROJECT STATE

Aquest document explica en quin punt es troba el projecte avui.

No serveix per descriure tot el projecte en detall. Serveix per donar context rapid abans de treballar en un canvi.

## Com omplir aquest document

Respon de forma curta:

- el projecte esta buit, en prototip o en desenvolupament real?
- que funciona ja?
- que falta encara?
- hi ha algun bloqueig important?

## Plantilla recomanada

### Status

Exemple:

- idea inicial
- prototip
- primera versio funcional
- en desenvolupament actiu

### Current situation

Explica en 3 o 5 linies que hi ha fet fins ara.

### What already works

- funcionalitat 1
- funcionalitat 2
- funcionalitat 3

### What is missing

- funcionalitat pendent 1
- funcionalitat pendent 2

### Known risks or blockers

- bloqueig tecnic
- dubte de producte
- limitacio de temps

## Exemple per a un estudiant de DAM

### Status

prototip inicial

### Current situation

Projecte d'una app web CRUD de tasques. Encara no hi ha autenticacio. La base del projecte esta creada, pero la major part de funcionalitats de negoci encara no existeixen.

### What already works

- estructura inicial del projecte
- routing basic
- connexio inicial amb base de dades o persistencia simple

### What is missing

- crear tasques
- editar tasques
- eliminar tasques
- validacions i revisio

### Known risks or blockers

- encara no esta tancada l'arquitectura final
- es prioritza simplicitat sobre escalabilitat
